

# AI 演示网站
✨ 一个现代化、响应式的 AI 体验与展示网站模板
 *（截图需补充）*
## 🚀 功能特性
- 响应式设计（适配手机/平板/桌面）
- 现代化UI设计（Material Design风格）
- 包含完整页面：
  - 🏠 主页：展示核心内容
  - 📖 关于页：个人/团队介绍 + 时间线
  - 🤖 AI 体验：接入 OpenAI 聊天接口
- 轻量级动画效果
- 暗黑模式支持（可选）
## 🛠️ 使用方式
1. 克隆仓库：
   ```bash
   git clone https://github.com/qwq202/beautiful-webpage.git
   ```
2. 直接编辑HTML/CSS文件
3. 替换`assets/images/`中的图片资源
## 🌐 部署
- GitHub Pages自动部署
- 或通过Netlify/Vercel部署
